# CloudDrive
* 本系统是一个基于javaweb的仿照百度网盘做的小型云盘系统 
* 前端使用**bootstrap**框架构建，后台未使用框架，使用的是最基本的*servlet*实现。
* 访问这个链接可查看[项目实际运行情况](http://47.107.160.68:8080/CloudDrive/)。
